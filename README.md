## About The Project

Demo Features:

* list - detail pages
* secondary layout with sidebar and header
* Listing page infinite scrolling
* ViewTransition based animation
* full responsive UI
* ~~That I am in too deep with Vercel+next+shadcn~~

Unimplemented features:

* full i18n support
